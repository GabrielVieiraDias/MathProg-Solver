About ...

MathProg is a Chrome application for linear and discrete optimization in GNU MathProg, the modeling language of the open source GLPK project. The syntax is similar to AMPL. Calculations are performed using the glpk.js solver.

The application uses code from a number of open source projects including

  • GNU GLPK project
  • glpk.js
  • jQuery
  • Bootstrap
  • MathJax
